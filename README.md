# Google form 웹앱

## 소개
Vanilla js로 구현한Google form 웹앱입니다.
라우터를 직접 구현하여 React의 SPA처럼 만들었습니다.
사용자의 여러가지 입력을 받아 데이터를 전달할 수 있습니다.


## 기능
- 입력들을 초기화 할 수 있습니다
- 브라우저 새로고침시에도 선택값들이 유지됩니다.
- 입력들을 선택하지 않으면 다음,이전화면으로 갈 수 없습니다.
- 제출하기를 누르면 지금까지 선택한 값들을 한곳에서 볼 수 있습니다.
